| 축 | 내용 |

| **도메인** | 누수음 탐지 / 배관 내부 진단 / 관망 분석 / 상수도 시스템 디지털화 |
| **기술** | 회로 · 펌웨어 · 기계설계 · Python 분석 · Jetson 엣지 · Unity AR · 로컬 LLM |
| **사업화** | 정부 R&D 30+ 과제 (TIPS / D-유니콘 / OI / K-water 등) 기획 · 수주 · 실행 |
| **글로벌** | 멕시코 · 키르기스스탄 · 탄자니아 · 태국 · 미국 · 인도네시아 — 6개국+ |

## 일하는 방식

- **한 사이클 완결** — 기획서 → 수주 → 회로 / 펌웨어 설계 → 분석 SW → 현장 설치까지 1인이 닫음
- **AI를 인프라로** — PPT / BOM / 리포트 자동생성, 분석 워크플로우 Python 화, 엣지 LLM 운영
- **도메인이 본업** — 자동제어 15년 + 수도 산업 깊이가 나머지 모든 기술의 방향타
