## 일하는 방식

- **한 사이클 완결** — 기획서 → 수주 → 회로 / 펌웨어 설계 → 분석 SW → 현장 설치까지 1인이 닫음
- **AI를 인프라로** — PPT / BOM / 리포트 자동생성, 분석 워크플로우 Python 화, 엣지 LLM 운영
- **도메인이 본업** — 자동제어 16년 + 수도 / 마이크로웨이브 산업 깊이가 나머지 모든 기술의 방향타

---

## 전문 분야 (직접 수행)

- 자동제어 시스템 설계 · **PLC (LS산전)** / **HMI (M2I, LS산전)** 프로그래밍
- 자동화 / 전기 판넬 제작 · 시운전 (현장)
- 전기설계 (S마크 기준) — BOM · 계통도 · 상세회로도 · 3D 외함 / 내판 (SolidWorks)
- 기구 보조 설계 — 제품 / 장비 외형 · 프레임 (SolidWorks)
- **아날로그 회로 설계** — 압전소자 센서 보드 / 누수음 신호 처리 (저잡음 / OPAMP)
- 누수음 탐지 · 신호 분석
- **마이크로웨이브 응용 설비** (1.1~3kW급, 11년) — 마그네트론 · 도파관 · HV Power Supply
- 해외 소형 정수장 염소 자동투입 장치 등 PLC / HMI 납품

## 주요 프로젝트 / 납품 이력

### ㈜위플랫 AI연구소 (2025 ~) — 상수도 디지털 트랜스포메이션 R&D

**진행 중**
- 상수도배관 증강현실 (Unity 기반 매설 배관 시각화)
- 통합관리시스템 (Jetson AGX Orin 엣지 디바이스)
- 스마트캡슐 R&D (배관 내부 영상 / 음향 수집 캡슐, 회수장치, 분해조립 지그)
- QGIS 상수도관망 분석기 (Python)
- Raspberry Pi 엣지 서버 (로컬 LLM)
- 정부 R&D — 딥테크 TIPS / 포스트 TIPS / OI 사업

**해외 사업**
- 태국 붕클라 (완료) — 소형 정수장 염소 자동투입 장치 PLC / HMI
- IBS 사업 (진행 중) — 해외 현장 시스템 개발
- 파이프라인 — 키르기스스탄 · 멕시코 · 탄자니아 · 미국 · 인도네시아

### ㈜글래스돔 코리아 (2023 ~ 2024) — PCF (Product Carbon Footprint) 모니터링 시스템 구축

- **조일알미늄 · 롯데알미늄** 탄소발자국 모니터링 시스템 구축 공사
- **하도급사 관리** · 현장 감독
- **PLC 기반 DAQ (데이터 수집)** 직접 수행
- 탄소국경조정(CBAM) 대응 인프라 구축 흐름

### ㈜한테크 (2022 ~ 2023) — 자동화 판넬 제작 / 시운전

- 전기판넬 제작 및 현장 시운전

### ㈜유림이엔지 → ㈜파워써플라이 → ㈜세종이엠씨 (2009 ~ 2022, 13년) — 마이크로웨이브 응용 설비
