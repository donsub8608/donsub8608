## 일하는 방식

- **도메인 지식이 본업** — 자동제어, 전기설계, 누수탐지, 계측제어 15년
- **AI로 실행 범위를 확장** — Python 분석, 펌웨어, Unity AR, OCR, 자동화 스크립트까지 AI 협업으로 직접 구현
- **결과 중심** — 외주 / 분업 없이 기획 → 설계 → 구현 → 현장 검증까지 한 사이클을 자체 완결
